Exercicio da Aula3: 19-09-2025, da disciplina de POO - 2º Periodo do curso de Sistemas para Internet - IFTO

"Faça um programa que utilize duas classes: Iniciar e Estudante

Na classe Iniciar deverá haver o método main.

Na classe estudante deverá haver os atributos de objeto:

nome, fone

Crie uma aplicação que gere 5 estudantes e depois imprima os dados de todos os estudantes na tela

- A partir de um método de classe da classe Iniciar - chamado imprimir

- A partir de um método de classe da classe Estudante chamado Imprimir

- A partir de um método de objeto da classe Estudante chamado Imprimir2"

Extras: Foi adicionado um Menu (Do-While e Switch-Case) e ArrayList para pratica de implemntação.
